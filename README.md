## 📌 Overview
StreamPulse is a lightweight browser extension built for the streaming community. It stays invisible until you need it, providing essential features for Twitch, Kick, and DLive without the bloat.

## ✨ Features
- 🔔 **Real-Time Alerts**: Get notified the exact second your favorite streamer goes live.
- 🪙 **Auto-Collect Channel Points**: Never miss Twitch channel points again, automatically claimed in the background.
- ⚡ **Optimized Latency**: Automatically refreshes players encountering errors (e.g., Twitch #2000 errors).
- 🧹 **Clean Interface**: Integrated dashboard, keyword filtering, and zero ads or tracking.

## 🛠️ Architecture
- **Manifest V3**: Fully compliant with the latest Chrome Web Store standards.
- **Service Workers**: Efficient background processing for notifications and API polling.
- **Vanilla JS & CSS**: No heavy frameworks, ensuring minimal memory footprint.

## 🚀 Installation for Development
1. Clone the repository:
   ```bash
   git clone https://github.com/AlexisAMZ/StreamPulse.git
   ```
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select this directory.
